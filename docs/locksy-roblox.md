# Roblox Module

### Settings

- `API_KEY`: This key is given to you when setting up the bot. You can either store the key as plain text here or as a secret

=== "Plain Text"

    ```lua
    API_KEY = "EXAMPLEKEY_1234567"
    ```

=== "Roblox Secret"

    ```lua
    API_KEY = game:GetService("HttpService"):GetSecret("API_KEY")
    ```
